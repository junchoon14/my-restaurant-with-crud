# 餐廳清單 :wink:
### Restaurant List Web
---
![restaurant](./s4a5q1.jpg)
---

1. **可以在首頁看到所有餐廳與它們的簡單資料：**
- [ ] 餐廳照片
- [ ] 餐廳名稱
- [ ] 餐廳分類
- [ ] 餐廳評分
  
2. **可以再點進去看餐廳的詳細資訊：**
- [ ] 類別
- [ ] 地址
- [ ] 電話
- [ ] 描述
- [ ] 圖片
 
3. **可以透過搜尋餐廳名稱來找到特定的餐廳**

4. **可以新增喜愛的餐廳**

5. **可以編輯現有餐廳**

6. **可以刪除不想要的餐廳**
